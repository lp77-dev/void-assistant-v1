Assistente híbrido de alto desempenho desenvolvido por **LP77**. Focado em automação local, segurança de sistema e inteligência offline.

1. Baixe o arquivo `setup.bat`.
2. Execute para configurar o ambiente isolado e baixar o motor de processamento.
3. Utilize o executável ou o atalho de inicialização para ativar o Void.
